泽PPT备份助手 更新日志 / Changelog
=================================

当前版本 / Current Version：v5.0

本项目当前发布标准备份版 / Standard Backup Edition。


v5.0 正式交付版 / Delivery Ready
--------------------------------
1. 增加“一键体检”，可查看版本、运行状态、备份目录、磁盘空间、日志和最近备份情况。
2. 增加“磁盘空间保护”，可限制每个备份位置最多占用 2GB / 5GB / 10GB / 20GB / 50GB。
3. 增加“备份格式”设置，可选择自动备份 PPT、Word、PDF 中的哪些类型。
4. 增加“备份位置”设置，可选择自定义备份目录，也可恢复默认 D/E/C 自动规则。
5. 增强主界面状态栏，显示最近扫描、今日备份、最近备份、启用格式、备份位置和空间保护状态。
6. 增加首次运行免责声明，提示用户仅在本人或已授权电脑上使用。
7. 设置会同时保存到软件目录和备份目录，提升自定义备份位置跨重启的可靠性。
8. “关于/反馈”窗口增加一键复制反馈邮箱。


v4.0 常用设置与客户交付完善 / Customer Delivery Improvements
-----------------------------------------------------------
1. 增加开机自启。
2. 增加备份成功提示开关。
3. 增加自动清理旧备份，可选择永久保留或保留最近 30 / 60 / 90 天。
4. 增加关于/反馈窗口。
5. 程序文件属性、窗口标题、托盘标题统一为“泽PPT备份助手”。
6. 发布包和教程命名改为客户可直接阅读的形式。
7. 使用教程改为简化版，并补充新功能用法。
8. 增加按版本号整理的更新日志。
9. 增加反馈邮箱：zeningshuyi@gmail.com。
10. 优化“删除选中”：源文件仍打开时，本次运行内不会马上重新备份同一个版本。


v3.0 多格式与产品整理 / Multi-format Improvements
------------------------------------------------
1. 从 PPT 扩展到 PPT / Word / PDF 自动备份。
2. 支持 Office、WPS 和常见 PDF 阅读器。
3. 增加浏览器 PDF 标题识别兼容。
4. 增加文件内容哈希去重。
5. 程序名称统一为“泽PPT备份助手.exe”。
6. 图标内置到程序中。


v2.0 主界面与稳定性增强 / Main UI and Stability
-----------------------------------------------
1. 增加右下角托盘后台运行。
2. 增加主界面，可查看最近备份记录。
3. 增加打开日志、打开备份目录、立即扫描功能。
4. 增加日志文件，方便查看运行情况。
5. 增加 D/E 盘不可用时的 C 盘和文档目录备用保存。
6. 增加删除记录、清理失效记录、清空记录、清空日志功能。


v1.0 基础备份版 / Basic Backup
------------------------------
1. 支持自动检测当前打开的 PPT 文件。
2. 支持将文件备份到 D 盘和 E 盘。
3. 支持按日期分类保存。
4. 支持重名文件自动编号，避免覆盖原文件。
