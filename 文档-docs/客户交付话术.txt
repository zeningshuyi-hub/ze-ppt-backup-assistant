泽PPT备份助手 客户交付话术
==========================

适用场景
--------
买家付款后、咨询软件怎么用、询问文件位置、找不到窗口、杀毒软件提示时，可以直接复制下面对应话术发送。


发货说明
--------
您好，软件已经发您。

使用方法很简单：
1. 解压压缩包。
2. 双击“泽PPT备份助手.exe”运行。
3. 第一次打开会显示免责声明，点“确定”即可。
4. 软件会在右下角后台运行。
5. 正常打开 PPT、Word、PDF 文件，软件会自动备份。

默认备份位置：
D:\泽宁PPPPPPPPTTTT备份\
E:\泽宁PPPPPPPPTTTT备份\

如果电脑没有 D/E 盘，会自动保存到 C 盘或文档目录。

右下角托盘图标双击可以打开主界面，主界面可以查看最近备份、打开备份目录、打开日志、一键体检和设置备份格式等。


买家问：软件怎么用？
--------------------
您好，解压后直接双击“泽PPT备份助手.exe”运行即可。

运行后软件会在右下角后台工作，您正常打开 PPT、Word、PDF 文件，它会自动复制一份到备份目录，不需要手动保存、不需要手动拖文件。

要查看记录的话，双击右下角托盘里的“泽PPT备份助手”图标，就能打开主界面。


买家问：文件备份在哪里？
----------------------
默认备份在：
D:\泽宁PPPPPPPPTTTT备份\
E:\泽宁PPPPPPPPTTTT备份\

里面会按日期自动分类，例如：
D:\泽宁PPPPPPPPTTTT备份\2026-04-19\

如果电脑没有 D/E 盘，软件会自动改存到 C 盘或当前用户的文档目录。

也可以打开软件主界面，点击“打开当前备份”，会直接打开当前实际使用的备份目录。


买家问：Wormhole网盘是什么？
---------------------------
这是一个临时传文件的辅助入口。

您在主界面选中某条备份记录后，点击“Wormhole网盘”，软件会帮您打开 Wormhole 网页，并在资源管理器中定位对应备份文件，同时复制文件路径。您手动把文件拖到网页里，就可以生成临时分享链接。


买家问：为什么右下角没窗口？
--------------------------
这个是正常的。

软件默认是后台运行，不会主动弹窗打扰使用。可以在右下角任务栏托盘区域找到“泽PPT备份助手”图标。

如果图标被系统隐藏了，可以点右下角的小箭头展开隐藏图标。双击图标就能打开主界面。


买家问：会不会联网？
------------------
标准版是本地备份工具，不需要账号、服务器或后台配置。

正常备份文件时不需要访问互联网，主要是在本机读取当前打开的文档路径，并复制到本地备份目录。


买家问：杀毒软件提示怎么办？
--------------------------
这个软件是本地备份工具，没有安装程序，也没有数字签名。部分杀毒软件或 Windows 安全中心可能会对个人开发的小工具提示风险，这是常见情况。

建议处理方式：
1. 确认文件来源是本店发送的压缩包。
2. 解压后放到固定目录运行。
3. 如果系统拦截，可以选择“更多信息”再点“仍要运行”。
4. 如果杀毒软件误报，可以把软件目录加入信任。

如果您介意任何安全提示，可以先不要运行，也可以联系我确认。


买家问：为什么没有自动备份？
--------------------------
请先确认：
1. 软件是否已经运行。
2. 右下角是否有“泽PPT备份助手”图标。
3. 打开的文件是否是本地 PPT、Word、PDF 文件。
4. 文件是否已经保存到本地磁盘。
5. 备份目录是否有写入权限。

您也可以打开主界面，点击“一键体检”，把体检内容复制发我，我帮您判断。


买家问：能不能只备份 PPT？
-------------------------
可以。

打开主界面，点击“备份格式”，只勾选 PPT，取消 Word 和 PDF 即可。


买家问：能不能改保存位置？
------------------------
可以。

打开主界面，点击“备份位置”，选择您想保存的目录即可。

如果以后想恢复默认，进入“备份位置”后留空，或点击“使用默认”。


买家问：怎么控制占用空间？
------------------------
打开主界面，点击“空间保护”，可以设置每个备份位置最多占用 2GB、5GB、10GB、20GB 或 50GB。

超过限制后，软件会优先清理最旧的备份文件。这个功能只清理备份文件，不会删除原文件。


买家问：怎么退出？
----------------
在右下角托盘找到“泽PPT备份助手”图标，右键点击“退出”即可关闭。
