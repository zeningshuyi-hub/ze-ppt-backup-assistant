泽PPT备份助手 使用说明
======================

当前版本：v5.0

产品简介
--------
泽PPT备份助手是一款用于自动留存课堂、会议、培训等场景中打开文档的轻量工具。

程序启动后会在后台静默运行，自动检测当前电脑打开的 PowerPoint、WPS 演示、Word/WPS 文字以及常见 PDF 文件，并将文件复制到本地备份目录。

自动备份过程不会弹窗打扰授课或演示。

第一次在当前设备运行时，程序会显示免责声明。用户确认后，后续正常启动不再重复提示。


支持格式
--------
.ppt
.pptx
.pps
.ppsx
.doc
.docx
.docm
.rtf
.wps
.pdf


适用场景
--------
1. 教室、会议室、培训室电脑课件留存。
2. 每天需要自动收集 PPT、Word、PDF 文件的环境。
3. 不方便要求使用者手动复制文件的电脑。
4. 需要减少人工拷贝、漏收文件、重名覆盖等问题的场景。


版本说明
--------
当前发布版本为标准备份版。

标准备份版负责本地自动备份、记录查看、日志查看、常用设置和售后体检，适合普通客户、朋友或无需服务器配置的电脑使用。


启动方式
--------
直接双击“泽PPT备份助手.exe”即可运行，无需安装。

程序启动后默认在右下角托盘后台运行，不主动弹出主窗口。

双击右下角“泽PPT备份助手”图标可以打开主界面。


备份位置
--------
程序会优先备份到以下目录：

D:\泽宁PPPPPPPPTTTT备份\日期\
E:\泽宁PPPPPPPPTTTT备份\日期\

如果 D 盘和 E 盘都不可用，会自动启用 C 盘备用目录：

C:\泽宁PPPPPPPPTTTT备份\日期\

如果 C 盘根目录没有写入权限，会退到当前用户文档目录：

C:\Users\当前用户\Documents\泽宁PPPPPPPPTTTT备份\日期\

示例：

D:\泽宁PPPPPPPPTTTT备份\2026-04-18\数学课件.pptx
E:\泽宁PPPPPPPPTTTT备份\2026-04-18\数学课件.pptx

如果同一目录下出现重名文件，程序会自动编号：

数学课件_2.pptx
数学课件_3.pptx


主界面
------
主界面功能包括：

1. 查看最近备份记录。
2. 打开日志。
3. 打开备份目录。
4. 打开当前实际使用的备份目录。
5. 立即扫描一次。
6. 设置开机自启。
7. 开启或关闭备份成功提示。
8. 设置自动清理旧备份。
9. 设置磁盘空间保护。
10. 选择自动备份格式。
11. 自定义备份位置。
12. 一键体检。
13. 查看关于和反馈邮箱。
14. 删除选中记录，并避免当前仍打开的同一版本文件马上重新备份。
15. 清理失效记录。
16. 清空记录。
17. 清空日志。

双击最近备份列表中的文件记录，会在文件资源管理器中定位到备份文件。


常用设置
--------
开机自启：
勾选“开机自启”后，程序会写入当前用户启动项。电脑开机并登录当前用户后，会自动启动泽PPT备份助手。

备份提示：
勾选“备份提示”后，每次成功备份文件，右下角会显示短提示。不勾选时，程序保持静默运行。

自动清理：
点击“自动清理”可以设置备份保留时间：

1. 永久保留，不自动删除。
2. 保留最近 30 天。
3. 保留最近 60 天。
4. 保留最近 90 天。

自动清理只会删除备份目录中按日期命名的旧文件夹，例如 2026-04-19。不会删除日志文件夹，也不会删除源文件。

磁盘空间保护：
点击“空间保护”可以限制每个备份位置最多占用空间。可选择不限制，或限制为 2GB / 5GB / 10GB / 20GB / 50GB。

超过限制后，程序会优先删除最旧的备份文件，并自动清理空日期文件夹。该功能只处理备份目录内按日期保存的备份文件，不会删除源文件。

备份格式：
点击“备份格式”可以选择自动备份 PPT、Word、PDF 中的哪些类型。至少需要保留一种格式。

备份位置：
点击“备份位置”可以选择自定义备份目录。设置自定义目录后，程序会优先备份到该目录；如果自定义目录不可用，会回退到默认 D/E/C 规则。

留空或点击“使用默认”，会恢复默认备份位置规则。

一键体检：
点击“一键体检”可以查看程序版本、运行状态、启用格式、备份目录写入情况、磁盘剩余空间、日志位置、今日备份数量、最近扫描状态和最近备份文件。

主界面状态栏：
主界面底部会显示当前运行状态，包括最近扫描时间、当前检测到的打开文件数量、今日备份数量、最近成功备份、启用格式、备份位置和空间保护状态。

关于/反馈：
点击“关于/反馈”可以查看产品信息和反馈邮箱，也可以一键复制邮箱。


日志与记录
----------
日志文件：

D:\泽宁PPPPPPPPTTTT备份\日志\泽.log

去重索引：

D:\泽宁PPPPPPPPTTTT备份\日志\backup-index.tsv

如果当前使用的是 C 盘或文档目录备用备份位置，日志和索引也会自动写入对应备用目录。

删除选中记录：
在主界面选中记录后点击“删除选中”，程序会删除对应备份文件，并移除去重记录。源文件不会被删除。如果源文件仍处于打开状态，本次运行内不会马上重新备份同一个版本；关闭并重新启动软件后，仍可再次备份。


去重规则
--------
程序会使用多种规则减少重复备份：

1. 路径、大小、修改时间完全相同：跳过。
2. 文件内容哈希相同：跳过。
3. 10 分钟内同名、同大小、同类型：跳过。

如果备份文件已被手动删除，程序会自动清理失效记录，并允许重新备份。用户主动点击“删除选中”后，同一个版本的源文件会在本次运行内临时忽略，避免刚删除又立刻出现。


PDF 检测说明
------------
PPT 和 Word 文件通过 Office/WPS 接口检测，稳定性较高。

PDF 没有统一接口，程序会检测常见 PDF 阅读器、WPS、Edge、Chrome 等进程命令行中的本地 PDF 路径。

部分阅读器不会暴露本地文件路径。程序会额外读取 QQ 浏览器窗口标题，并尝试在桌面、下载、文档、D/E 盘和 U 盘中查找同名 PDF。

如果某个 PDF 阅读器既不暴露路径，窗口标题也不是文件名，可能无法自动检测。遇到这种情况，建议使用 WPS、Edge、Adobe Reader 或 Foxit 打开 PDF。

注意事项
--------
1. 冰点还原或还原卡环境下，请将程序和备份目录放在不会被还原清空的盘符。
2. 如果 E 盘不存在，程序会自动跳过 E 盘。
3. 如果 D/E 都不可用，程序会自动使用 C 盘或文档目录备用位置。
4. 标准备份版不需要账号、服务器或后台配置。
5. 备份文件可能包含个人或工作资料，请妥善保管备份目录。
