# v5.0 发布说明

这是泽PPT备份助手的正式交付增强版。

## 推荐分发文件

`泽PPT标准备份版.zip`

标准备份版只做本地自动备份，不自动上传文件，适合普通客户使用。

## v5.0 重点更新

- 增加一键体检，便于客户售后排查
- 增加磁盘空间保护，可限制备份目录占用空间
- 增加备份格式设置，可选择 PPT / Word / PDF
- 增加备份位置设置，可自定义目录或恢复默认规则
- 增强主界面状态栏，显示最近扫描、今日备份、备份位置等信息
- 增加首次运行免责声明
- 关于/反馈窗口支持一键复制邮箱
